تمرین دوم WebGIS
درس GIS تحت وب _ دانشگاه صنعتی خواجه نصیرالدین طوسی _ محیا ساعدی 

--------------------------------------------------

توضیح پروژه
این پروژه به‌عنوان تمرین دوم درس GIS تحت وب انجام شده است.
هدف تمرین: پیاده‌سازی یک برنامه WebGIS سمت کاربر با استفاده از JavaScript و سرویس‌های تحت وب است.

در این پروژه یک نقشه تعاملی نمایش داده می‌شود که کاربر می‌تواند:
- یک مکان را جستجو کند.
- یا با کلیک روی نقشه، اطلاعات هواشناسی آن نقطه را مشاهده کند.

--------------------------------------------------

امکانات پروژه
- نمایش نقشه تعاملی با استفاده از کتابخانه OpenLayers
- جستجوی مکان (Geocoding)
- دریافت مختصات جغرافیایی با کلیک روی نقشه
- نمایش اطلاعات هواشناسی شامل:
  • دما
  • وضعیت هوا
  • رطوبت

--------------------------------------------------

ابزارها و فناوری‌های استفاده‌شده:
- HTML
- CSS
- JavaScript
- OpenLayers
- OpenWeather API

--------------------------------------------------

نحوه اجرای پروژه:
1. پروژه از گیت‌هاب Clone می‌شود.
2. فایل config.example.js به config.js تبدیل می‌شود.
3. کلید API مربوط به OpenWeather داخل فایل config.js قرار می‌گیرد.
4. فایل index.html در مرورگر اجرا می‌شود.

--------------------------------------------------

API هواشناسی
برای دریافت اطلاعات هواشناسی از OpenWeather API استفاده شده است.
داده‌های هواشناسی بر اساس مختصات جغرافیایی (Latitude و Longitude)
که از کلیک روی نقشه به دست می‌آید، دریافت می‌شوند.

--------------------------------------------------

مدیریت کلید API
به دلیل مسائل امنیتی:
- کلید API داخل فایل config.js قرار داده شده است.
- این فایل در .gitignore اضافه شده و در مخزن گیت‌هاب Commit نمی‌شود.

به این ترتیب کلید API فقط به‌صورت محلی استفاده می‌شود
و در مخزن عمومی قرار نمی‌گیرد.

--------------------------------------------------

مقایسه APIهای هواشناسی

OpenWeather API
- نسخه رایگان: دارد
- نیاز به ثبت‌نام: دارد
- سادگی استفاده: بالا
- کیفیت مستندات: کامل و واضح
- پشتیبانی از مختصات جغرافیایی: دارد
- مناسب برای پروژه دانشجویی: بسیار مناسب

WeatherAPI
- نسخه رایگان: دارد (با محدودیت بیشتر)
- نیاز به ثبت‌نام: دارد
- سادگی استفاده: متوسط
- کیفیت مستندات: خوب
- پشتیبانی از مختصات جغرافیایی: دارد
- مناسب برای پروژه دانشجویی: نسبتاً مناسب

Meteostat
- نسخه رایگان: دارد
- نیاز به ثبت‌نام: ندارد
- سادگی استفاده: پایین
- کیفیت مستندات: فنی و پیچیده
- پشتیبانی از مختصات جغرافیایی: دارد
- مناسب برای پروژه دانشجویی: کمتر مناسب

--------------------------------------------------

نتیجه مقایسه و انتخاب API
با توجه به مقایسه انجام‌شده، OpenWeather API به دلیل داشتن نسخه رایگان مناسب،
سادگی استفاده، مستندات کامل و سازگاری بالا با پروژه‌های WebGIS دانشجویی،
به‌عنوان API نهایی این پروژه انتخاب شد.

--------------------------------------------------

نتیجه‌گیری
در این پروژه یک WebGIS ساده ولی کاربردی پیاده‌سازی شد که ترکیبی از نقشه، جستجوی مکانی و داده‌های هواشناسی را در اختیار کاربر قرار می‌دهد.

این پروژه مفاهیم اصلی GIS تحت وب، کار با APIها و ارتباط بین نقشه و داده‌های مکانی را به‌صورت عملی پیاده‌سازی و نمایش می‌دهد.
